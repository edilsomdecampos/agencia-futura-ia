# 🛸 Agência de IA Futurista | Next-Gen Interface

Esta é uma landing page de alto impacto desenvolvida para representar a vanguarda da Inteligência Artificial. O projeto foca na imersão do utilizador através de elementos visuais dinâmicos e uma estética "Cyber-Luxury".

## 🚀 Diferenciais de Engenharia e Design

- **Experiência Imersiva:** Background dinâmico com vídeo 4K (`nave.mp4`) integrado via CSS para carregamento fluído.
- **Identidade Visual High-Tech:** Utilização da tipografia *Orbitron* (Google Fonts) e efeitos de brilho neon com camadas de `text-shadow`.
- **Animações Estratégicas:** Implementação de `keyframes` para revelação progressiva (fade-in) de conteúdos, criando um ritmo visual sofisticado.
- **Design Responsivo:** Interface adaptável que mantém o impacto visual em diferentes resoluções.

## 🛠️ Stack Tecnológica
- **HTML5:** Estrutura semântica para SEO e acessibilidade.
- **CSS3 Avançado:** Filtros de brilho, animações cronometradas e manipulação de vídeo em background.
- **Hospedagem:** Deploy automatizado via Vercel (Production Ready).
